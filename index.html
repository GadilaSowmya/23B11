Understanding Search Algorithms

Search algorithms help find an element in a collection of data. Here, weâ€™ll explore two common search techniques: Linear Search and Binary Search.

Linear Search

Linear search sequentially checks each element until the target is found or the list ends. It works on unsorted data.


function linearSearch(arr, target) {
    for (let i = 0; i < arr.length; i++) {
        if (arr[i] === target) return i;
    }
    return -1;
}

Binary Search

Binary search works on sorted data. It divides the search space in half until the target is found.


function binarySearch(arr, target) {
    let low = 0, high = arr.length - 1;
    while (low <= high) {
        let mid = Math.floor((low + high) / 2);
        if (arr[mid] === target) return mid;
        else if (arr[mid] < target) low = mid + 1;
        else high = mid - 1;
    }
    return -1;
}
